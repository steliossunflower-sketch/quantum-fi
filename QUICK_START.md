# Quick Start Guide - 5 Minute Setup

Want to get started RIGHT NOW? Follow these steps:

## 1. Create GitHub Account (2 minutes)
- Go to github.com
- Sign up (free)
- Done!

## 2. Upload Your Code (3 minutes)
- Click "+" > "New repository"
- Name it: `golden-ratio-pulse`
- Make it Public
- Click "uploading an existing file"
- Drag ALL files from this folder
- Click "Commit changes"
- Done! Your code is online!

## 3. Share Your GitHub Link
- Copy your repository URL
- Share it on:
  - Twitter/X
  - Reddit (r/Python, r/quantum)
  - LinkedIn
  - With friends/family
