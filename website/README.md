# Website for Golden Ratio Quantum Pulse Project

This is a simple, professional website showcasing the Golden Ratio Quantum Pulse discovery.

## Files

- `index.html` - Main webpage
- `styles.css` - Styling for the website
- `README.md` - This file

## Deployment to GitHub Pages

1. Make sure your GitHub repository is set up
2. Go to repository Settings > Pages
3. Select source: "Deploy from a branch"
4. Choose branch: "main" (or "master")
5. Select folder: "/website" or "/ (root)"
6. Click Save

Your website will be available at: `https://yourusername.github.io/quantum-fi/`

## Customization

Before deploying, update these in `index.html`:
- Replace `yourusername` with your actual GitHub username in the GitHub links
- Update any personal information
- Add your actual GitHub repository URL

## Local Testing

1. Run `python pulse_comparison.py` from the project root to generate the comparison image (saved to both project root and `website/`)
2. Open `website/index.html` in a web browser to preview
